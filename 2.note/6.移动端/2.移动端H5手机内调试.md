::: tip
移动端H5手机内调试 神器：vconsole
:::

## 🥞第一部分：vconsole使用

### 基本使用

```html
<!-- CDN引入 -->
<script src="https://wechatfe.github.io/vconsole/lib/vconsole.min.js?v=3.2.0"></script>
<script>
    new VConsole();
</script>
```

- [vcondole npm 地址](https://www.npmjs.com/package/vconsole)

**功能：**

- 查看 console 日志
- 查看网络请求
- 查看页面 element 结构
- 查看 Cookies、localStorage 和 SessionStorage
- 手动执行 JS 命令行
- 自定义插件
