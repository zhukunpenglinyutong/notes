## 🎖使用说明

```sh
# 克隆
git clone https://github.com/zhukunpenglinyutong/notes.git

# 安装依赖（注意，一定要yarn安装，并注意不要删除 yarn.lock）
yarn

# 预览，VuePress会启动一个预览的网址
npm run start

# 打包，打包后资源在 docs/.vuepress/dist下
npm run build
```
