## 🎖使用说明

```sh
# 克隆
git clone https://github.com/zhukunpenglinyutong/notes.git

# 安装依赖
npm install

# 预览，VuePress会启动一个预览的网址
npm run start

# 打包，打包后资源在 doc/.vuepress/dist下
npm run build
```

**构建说明，为什么首页的几个文件夹，就能生成网站？而不用配置VuePress？详细请见 doc/.vuepress/vueperssEs文件夹**
